Ninja-error-log
===============
Ninja-Error-Log is a plugin for NINJA-IDE. Allows you to view and delete the file Ninja.exe.log (Only for Windows)

version: **0.1**<br />
Licensed under: **GPLv3**<br />
Ninja-IDE version: **>= 2.3**

--------------
